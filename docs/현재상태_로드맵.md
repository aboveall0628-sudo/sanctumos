```
                                  ↓
                              [소크라테스 흐름]
                                  ↓
                              [B-2 원칙 계층] ──→ [CS AI]

[B-4 통합 트랙]
   ├─ 본인 프로필
   ├─ 인물 카드 보강 (두 면)

```
