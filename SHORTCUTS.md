# Sanctum OS 단축키

키보드만으로 거의 모든 동작을 처리할 수 있도록 만들었어요.
앱 안에서 언제든 **`Ctrl + /`** 를 누르면 같은 목록이 검색 가능한 모달로 떠요.

> 환경: Windows + QWERTY 기준. 브라우저·OS 기본 단축키(`Ctrl+C/V/X/Z/Y/A/F/R/T/W/Tab`, `Alt+Tab`, `F5/F11` 등)는 그대로 둬요.

---

## 어디서나 (전역)

| 단축키 | 동작 |
|---|---|
| `Ctrl` + `/` · `?` | 단축키 도움말 켜고 끄기 |
| `Ctrl` + `,` | 설정 화면 열기 |
| `Ctrl` + `S` | 수동 저장 + 저장됨 토스트 (브라우저 기본 가로채기) |
| `Esc` | 모달·패널 닫기 · 입력에서 빠져나가기 |
| `Ctrl` + `K` | 커맨드 팔레트 *(다음 단계 예정)* |
| `Alt` + `←` / `Alt` + `→` | 앱 안 이전 / 다음 화면 |
| `Ctrl` + `Shift` + `L` | 즉시 잠금 |
| `Ctrl` + `Shift` + `H` | 화면 가리기 토글 |
| `Ctrl` + `Shift` + `D` | 다크 · 라이트 모드 토글 |
| `Ctrl` + `Shift` + `B` | 백업 화면 열기 |

---

## 화면 이동

숫자로 (위치) 와 문자로 (의미) 둘 다 돼요.

| 화면 | 숫자 | 문자 |
|---|---|---|
| 대시보드 | `Alt` + `1` | `Alt` + `D` |
| 오늘 | `Alt` + `2` | `Alt` + `T` |
| 지난 묵상 | `Alt` + `3` | `Alt` + `M` |
| 나의 원칙 | `Alt` + `4` | `Alt` + `P` |
| 리포트 | `Alt` + `5` | `Alt` + `R` |
| 인물 | `Alt` + `6` | `Alt` + `U` |
| 조직 | `Alt` + `7` | `Alt` + `O` |
| 설정 | `Alt` + `8` | `Alt` + `S` |

---

## 글쓰기 칸 안에서 (묵상 노트, 입력 필드)

| 단축키 | 동작 |
|---|---|
| `Ctrl` + `Enter` | 저장 · 제출 (글쓰기 칸을 빠져나오며 자동 저장 트리거) |
| `Ctrl` + `S` | 수동 저장 + 저장됨 토스트 |
| `Esc` | 입력에서 빠져나가기 (자동 저장 트리거) |

> 글쓰기 칸 안에서는 단일 키 단축키(`?`, `J`, `K`, `N` 같은)는 **자동으로 꺼져요** — 글이 의도치 않게 가로채일 일이 없어요.

> 줄/블록 단위 이동(`Alt+↑↓`), 줄 삭제(`Ctrl+Shift+Backspace`), 슬래시 `/` 블록 메뉴, `@` 멘션, `Ctrl+B/I` 서식은 묵상 에디터 자체를 재설계하는 **별도 트랙**에서 함께 들어와요.

---

## 모달 · 다이얼로그 안에서

| 단축키 | 동작 |
|---|---|
| `Esc` | 닫기 · 취소 |
| `Tab` / `Shift` + `Tab` | 모달 안에서만 포커스 순환 (밖으로 안 나감) |

모달 밖 영역(백드롭)을 클릭해도 닫혀요.

---

## 안전 정책

- **브라우저·OS 기본은 안 건드려요.** `Ctrl+C/V/X/Z/Y/A/F/R/T/W/Tab`, `Alt+Tab/F4`, `Ctrl+Shift+T`, `F5/F11` 그대로 작동.
- **위험한 동작은 한 번 더.** 잠금·화면 가리기·테마·백업 같은 전역 동작은 `Ctrl+Shift+키` 조합으로만 발동.
- **잠금 화면에선 비활성.** 비밀번호 입력 중엔 앱 단축키가 모두 꺼져요.
- **한글 입력 중엔 무시.** IME 조합 중일 땐 어떤 단축키도 발동하지 않아요.
- **설정에서 끌 수 있어요.** [설정] → "키보드 단축키" 카드에서 전체 또는 단일 문자 단축키만 따로 끌 수 있어요. WCAG 접근성 정책.

---

## 도움이 안 될 때

- 단축키가 안 먹어요 → 입력 칸에 포커스가 있나 확인. `Esc` 한 번 누른 다음 다시 시도.
- 어디 있는지 모르겠어요 → `Ctrl + /` 누르고 검색.
- 다 꺼버리고 싶어요 → `Alt + 8` 로 설정 가서 "단축키 사용" 토글 끄기.
